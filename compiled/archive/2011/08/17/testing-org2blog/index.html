测试org2blog

因为看到 Planet Emacs 上不少比较活跃的 blogger 都开始用 org2blog 来 写作和发布 blog，我也决定试一把。org2blog 的安装很简单，在 wordpress 里则要注意打开 XML-RPC 功能。这篇 blog 用了 emacs 的 auto-fill，不知道转换出来以后是否还正常。

后来发现7.01的 org 似乎还不能满足 org2blog 的要求，所以直接升到了 git 开发版。现在再试一试，应该就没有问题了吧？
